vesicle results¶

Input Data¶

We recommend training using AC4 and evaluating using AC3. The provided vesiclerf code will download these volumes automatically. More information about the available data, including computer vision membranes and vesicles can be found in the datamap.

View Results¶

For the most interactive browsing experience, try using OCPViz. We recommend changing the coordinates and sliders as shown in the following picture as a starting point.

To easily view the results, you may wish to execute RESTful calls of the following form in your browser. Click here to download Figure 5 from the paper.
